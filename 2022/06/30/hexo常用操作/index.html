hexo 发布

　　hexo 的发布目前来说就只是使用到三个命令分别是：hexo generate 或者说是 hexo g 用于生成网页内容，然后再使用 hexo server 或者说是 hexo s 来通过本地服务器的4000端口来展示渲染之后的网站内容，最后再通过 hexo deploy 或者是 hexo d 来将网页推送到远程服务器上。
